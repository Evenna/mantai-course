# 曼塔 AI 体验课 PPT · 图片生成清单

> 用途：本套 PPT（3.2 团队介绍往后）中所有需要配图的位置，已放好占位图（SVG）。
> 你只需拿下面的 prompt 去 Midjourney / Flux / 即梦 / Seedream 等工具生成真实图片，
> 命名成对应文件名，覆盖 `ppt/assets/` 里的占位图即可，无需改代码。
>
> **⚠️ 本套图分两种风格，别混：**
> - **A 明亮设计风**（封面 / 团队背景 / 理想 / 工具 / CTA + 4 个人物）→ 明媚可爱、简约高级、时尚设计感。
> - **B 完全写实风**（参赛拿奖 / 接单赚钱 那一组）→ 真实摄影照片，越真实越好，用来证明"真能拿奖、真能接单赚钱"。

## 备注
- A 明亮风图建议 1920×1080（16:9），人物 800×1000（4:5）。
- B 写实图同样 1920×1080 或更高，越真实越好，用来撑可信度。
- 人物头像（teacher-1/2/3）如果想更可信，也可换成老师**真实职业照**，裁 4:5 竖构图。
- professor / mission / cta / tool 用意象/卡通图即可，无需真人，规避肖像风险。
- award / money 这一组**务必用写实照片**——这是"真能拿奖、真能赚钱"的证据，插画会削弱说服力。
