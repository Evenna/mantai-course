# 曼塔 AI · Agent 体验课 · 完整讲师脚本

> 面向：19.9 元体验课学员（零基础 / 好奇型 / 想搞副业型）
> 时长：全程约 60—75 分钟（Part1 安装 15min + Part2 实操 25min + Part3 案例与引导 20min）
> 讲师风格提示：**少讲概念、多让学员动手、每一步都有"啊原来这么简单"的爽点**。本脚本从 Part2 开始（安装 Part1 你已自备）。

---

## 【承上】Part 1 结束 → Part 2 过渡（30 秒）

> 讲师说：

"好，到这一步，你的 Agent 已经装好、控制台也登录了，第一条任务也跑通了——恭喜你，**你已经比 90% 只听过 AI Agent、但从没真正用过的人走得更远了**。

但装好只是拿到了工具。今天真正好玩的部分现在才开始：**接下来 20 分钟，我们不看 PPT、不背提示词，我带你亲手做出你人生中第一个 AI Agent 项目。**

我先说好——你什么代码都不用写，你只需要**用大白话跟它说你想要什么**。准备好了吗？打开你的 Agent 输入框，跟我一起来。"

---

## 【Part 2】手把手做出第一个 Agent 项目（约 25 分钟）

### 设计思路（讲师内部说明，不念出来）

体验课的黄金法则：**第一个项目必须 3 分钟内出成果、结果要"能晒能用"、过程要暴露 Agent 的"超能力时刻"。**

不要选"写个 Excel 公式"这种平淡任务。我们选一个能让学员**当场截图发朋友圈**的项目——做一个**属于自己的、能在手机上打开的个人网页**（Vibe Coding 最直观的爽点）。

备选方向（讲师根据学员画像二选一）：
- **A｜主推：做一个"个人介绍/作品名片"网页**——通用、人人有共鸣、成果可分享。
- **B｜内容/运营人群：小红书爆款选题+文案工作流**——贴近副业场景，衔接 Part3 赚钱话题。

下面主推 A，B 作为备用在文末附。

---

### 🎬 环节 2.1 — 抛出目标，制造期待（2 分钟）

> 讲师说：

"我们要做的第一个东西，是**一个完全属于你自己的个人网页**——不是填模板，是从零'聊'出来的。你告诉它你是谁、喜欢什么风格，它当场给你写出一个能在手机上打开、能发给朋友看的网页。

很多人第一次做完的反应是：'就这？我啥都没干它就做完了？'——对，这就是 Agent 和你以前用的 AI 聊天工具最大的不同：**它不只是回答你，它真的动手帮你把东西做出来了。**

来，第一句话跟我打——"

---

### 🎬 环节 2.2 — 第一次对话：让它动手（5 分钟）

> **让学员在输入框输入（讲师带读，鼓励改成自己的信息）：**

```
帮我做一个个人主页网页。我叫[名字]，是一名[职业/学生身份]，
喜欢[两三个兴趣]。风格要[简约现代 / 科技感 / 温暖治愈]，
深色背景，要有一个大标题、一段自我介绍、和我的联系方式。
做成一个可以直接在浏览器打开的 HTML 文件。
```

> 讲师说（学员输入时）：

"注意看它接下来做什么——它**不会只给你一段文字建议**，它会真的开始**创建文件、写代码**。你会看到它在动，就像有个程序员在你旁边帮你干活。这个过程我们叫 **Vibe Coding**——你负责'想要什么'，它负责'怎么实现'。"

> **成果出现后，讲师引导：**

"看到那个文件了吗？我们让它**直接打开给你看**——你再输入一句："

```
在浏览器里打开这个网页给我看看
```

"哇——这是**你的**网页。三分钟前它还不存在，现在它是一个真实的、能打开的作品。这就是你的第一个 Agent 项目。"

> 🎯 **爽点提示（讲师务必停顿，让学员消化）：**
> "我想请你现在做一件事：截个图。因为这是你和 AI 一起做出的第一个东西，值得记住这一刻。"

---

### 🎬 环节 2.3 — 迭代修改：暴露 Agent 的真正威力（6 分钟）

> 讲师说：

"但真正让我觉得 Agent'恐怖'的地方还没到。你不满意？没关系，你**不用重做，直接跟它说改哪里就行**。来，我们改几次——"

> **依次让学员输入（每次等结果出来再进行下一句）：**

```
把标题字体改大一点，加一个打字机的动画效果
```
```
背景加一点会动的粒子，鼠标移动的时候有反应
```
```
帮我加一个"作品集"板块，先放三个占位卡片
```

> 讲师说：

"感受到了吗？**每一次你只说'我想要什么'，它就帮你改好了。**你不需要知道什么叫 CSS、什么叫 JavaScript、什么叫动画曲线——这些以前需要程序员花几小时的活儿，你用一句话、几秒钟就完成了。

这就是为什么我们说：**AI Agent 时代，会表达想法的人，就是能造东西的人。**"

---

### 🎬 环节 2.4 — 小结与钩子（2 分钟）

> 讲师说：

"来回顾一下你刚才做了什么：
- ✅ 从零做出一个真实的个人网页
- ✅ 学会了'我说想法，它来实现'的协作方式
- ✅ 体验了'改哪说哪'的迭代威力

这只是一个静态网页。你可能已经在想——'那它还能做什么？' 别急，我先给你看**别人用同样的方式做出了什么惊人的东西**，看完你就知道你今天迈出的这一步，通向的是一个多大的世界。"

> （自然过渡到 Part 3）

---

### 📎 备用方向 B｜小红书爆款工作流（面向内容/副业人群）

> 让学员输入：

```
我想做小红书，主题方向是[美食探店/穿搭/职场干货]。
帮我策划一条爆款笔记：给我 5 个选题，选一个最好的，
写出标题、正文文案、话题标签，再给我配图建议。
```

> 讲师说：
"看——它一次性给你**选题、标题、文案、标签、配图思路**全套。你以前要花一下午想的东西，它几十秒就给你一整套可直接发布的内容。这就是很多学员后来靠它接单赚钱的起点——记住这个感觉，我们 Part3 会讲怎么把它变成收入。"

## 【Part 3】优秀案例 + 背书 + 资源引导报名（约 20 分钟）

> **本部分核心目标：** 从"你也能做" → "牛人用它做出了什么" → "我们凭什么带你" → "学了能赚钱/拿奖" → "所以你该报正课"。情绪曲线：**惊叹 → 信任 → 心动 → 行动。**

---

### 🎬 环节 3.1 — 优秀案例展示：打开天花板（8 分钟）

> 讲师说：

"你刚才做的是第一个网页。现在我给你看，**当有人把这套方法用到极致，能做出什么。** 这些**全部是用 AI 对话写出来的真实上线作品**，很多拿了国际设计大奖。打开这个页面，我们一个个看——"

> **打开：https://evenna.github.io/vibe-coding-showcase/**（讲师边滚动边讲，不用每个都念，挑 3—4 个高光讲）

---

#### 💥 高光案例①（最震撼开场）— Pieter Levels 飞行模拟器

> 讲师说：

"先看这个。独立开发者 Pieter Levels 对 AI 说了一句话：**'做一个飞行模拟器'**。几个小时后——一个**能多人联机、有空管对话、有天气云层地形的完整 3D 飞行游戏**上线了，全网 100 万+ 曝光。

作者自己都说：'我从头到尾没完整看过一遍代码。' **这就是 Vibe Coding 最出圈的封神时刻**——不是他多会写代码，是他**会指挥 AI**。"

---

#### 💥 高光案例② — Bruno Simon 驾车作品集（Awwwards 年度站点）

> 讲师说：

"再看这个。打开网页，你不是在'浏览',你是在**开一辆小车**在 3D 世界里穿梭，撞倒障碍、开到每个停靠点看他的作品。10M+ 浏览量，拿了 Awwwards 年度最佳。**它彻底重新定义了'简历/作品集'长什么样。**"

---

#### 💥 高光案例③ — Nithin Warrier（明确标注 Claude Code 构建）

> 讲师说：

"这个更关键。主页上**大大方方写着一行字：『brought to life with Claude Code』**——就是用 AI 对话做出来的。逐字拼出名字的打字动效、3D 首屏、悬停动画……**它是首批登上国际设计大奖的 AI 编写网站之一。** 也就是说，你今天学的这套方法，做出来的东西是可以拿去参赛拿奖的。"

---

#### 💥 高光案例④（自己人 · 建立亲近感）— Evans 手势流体（清华美院）

> 讲师说：

"最后这个，是**我们自己团队的作品**。清华美院的作品——打开摄像头，用**双手在空中挥动**，十根手指各自拖出一束彩色流体，零按钮、零学习成本。这是清华在做的'分寸感引擎'研究原型。

我放这个不是炫技，是想告诉你：**从你今天做的第一个网页，到清华在做的前沿交互研究，中间不是一道天堑，是一条你今天已经踏上的路。**"

> 🎯 **收束（讲师停顿）：**
> "这些人和你的区别，不是智商、不是学历、更不是会不会写代码。**唯一的区别是他们比你早开始、有人带、坚持做下去了。** 而'有人带'这件事——就是我们接下来要聊的。"

---

### 🎬 环节 3.2 — 背书与信任：我们凭什么带你（4 分钟）

> 讲师说：

"你可能会想：网上教 AI 的人那么多，凭什么跟我们学？我给你三个实打实的理由——不吹，都是可以查证的。"

> **① 名校大厂背书（讲师逐条说，语气笃定）：**

"第一，**团队背景**。我们团队来自**清华、腾讯、字节**——不是'合作过'，是真的从这些地方出来的人在带你。做这门课的核心成员，有前字节、前微信 AI 的专家。"

> **② 清华合作 + 国际工作坊 + 学术评委：**

"第二，**我们不是只会做课的团队，是真的在一线做教育和研究**。我们和**清华合作开展过 Design Future 国际工作坊**，教大家用这些 AI 工具做设计；办过**线下体验课**，还请了**北理工的教授来做评委**。也就是说——这套教学方法，是经过学术场景和真实学生验证过的，不是拍脑袋编的。"

> **③ 真实作品:**

"第三，你刚才看到的案例页里，就有**我们自己团队的清华美院作品**。**我们自己就在用这套方法做出拿得出手的东西**——不是只教你，是我们自己也在做。"

> 🎯 讲师收束：
> "所以你跟我们学，学的不是'网上随便一个博主总结的技巧'，而是**清华+大厂的一线打法，经过国际工作坊和高校评委验证过的体系。**"
