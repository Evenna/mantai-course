# 【Part 2】手把手做出你的第一个 Agent 项目

## 2.0 过渡开场

【轻松点，像跟朋友聊天】

"好，装到这儿，你的 Agent 已经能用了，控制台也进去了，第一条任务也跑通了。

其实吧，光是走到这一步，你就已经超过挺多人了。"AI Agent"这个词大家都听过，刷抖音刷朋友圈天天看到，但真正自己动手装好、用起来的人，其实没几个。大部分人都停在"听说过"，你现在已经"用起来了"，这不一样。

不过话说回来，装好只是拿到工具而已。就跟你买了台相机一个道理，买回来不是重点，拍出好照片才是。所以接下来这二十分钟，才是今天真正好玩的地方。

我先跟你说清楚：接下来你不用写一行代码，也不用记什么专业词。你就把它当成一个特别能干的助理，用大白话告诉它你想要啥，然后看着它帮你做出来就行。

来，把输入框打开，咱们做你的第一个东西。"

【停顿，等大家打开输入框】

## 2.1 抛出目标

"咱们第一个要做的，是一个属于你自己的个人主页网页。

不是那种填模板换头像的。是真从一张白纸开始，你告诉它你是谁、想要什么感觉，它当场给你写一个能在浏览器打开、能直接发给朋友看的网页。

我带学员做到这儿，好多人做完都愣一下，说"就……就好了？我啥也没干啊"。

【笑】

对，就好了。这也是 Agent 跟你以前用的那些聊天 AI 最不一样的地方——

以前的 AI，你问它，它回你一段字，剩下的活还得你自己干。Agent 不是，它是真动手帮你把东西做出来。它会建文件、写代码、打开给你看。一个是"会聊天的"，一个是"会干活的"，差别就在这儿。

来，第一句话，跟我一起打。"

## 2.2 第一次对话

【放慢，带大家一起输入，屏幕打出示范】

> ```
> 帮我做一个个人主页网页。我叫[你的名字]，是一名[你的职业或学生身份]，
> 平时喜欢[两三个兴趣]。风格我想要[简约现代 / 科技感 / 温暖治愈]，
> 深色背景，页面上有个大标题、一段自我介绍、还有联系方式。
> 做成一个能直接用浏览器打开的 HTML 文件。
> ```

"把里面的名字、身份、爱好换成你自己的，填好发出去就行。

【学员输入时引导看屏幕】

现在盯着屏幕看它干啥——它不会只回你一段字，你会看到它真的动起来了，在建文件、在一行行写代码。就像有个程序员坐你旁边，你说完一句，他埋头就开始敲。

这个过程有个词，叫 Vibe Coding。说白了就是：你负责说想要什么，它负责搞定怎么实现。你出主意，它干活。"

【等生成完成】

"好，做完了。咱们让它直接打开看看，再打一句——"

> ```
> 在浏览器里打开这个网页给我看看
> ```

【网页打开时】

"看——这就是你的网页。你的名字、你的介绍，都在上面了。

你想想，几分钟前这东西还不存在呢，你几句话就把它做出来了。这就是你的第一个作品。

我建议你截个图存着。倒不是为了发朋友圈——是这玩意儿是你跟 AI 一起做的第一个东西，留个纪念，挺有意义的。"

## 2.3 迭代修改

【"好戏在后头"的语气】

"不过说实话，刚才这个还不算最带劲的。真正让我第一次用的时候有点上头的，是改东西。

哪儿不满意？你不用重做，也不用去碰代码，直接跟它说改哪儿就行。咱们改几次你感受下。

先让标题更抓眼——"

> ```
> 把标题字体调大一点，加个打字机逐字打出来的动画
> ```

【等结果】"看，字变大了，还一个个蹦出来。接着，让背景动起来——"

> ```
> 背景加点慢慢飘的粒子，鼠标动的时候它有反应
> ```

【等结果】"背景活了。最后加个作品集的位置——"

> ```
> 加一个"作品集"板块，先放三个空卡片，我以后往里填
> ```

【等结果，稍微认真点】

"停一下，你感受下刚才发生了啥。

打字机动画、会动的粒子、能交互的卡片——这几样要是找个真人程序员做，起码好几个小时。你呢？每次就说一句想要什么，几秒就好了。

所以我特别想跟你说一句话：以前是会写代码的人才能造东西，现在，是会把想法说清楚的人就能造东西。这个变化，比你想的要大。"

【停顿】

## 2.4 加做第二个项目：用 AI 做小红书运营

【转场，语气自然】

"刚才那个是做"东西"。但你可能会想——我又不是设计师，做个网页对我有啥用？

行，那我给你换个更接地气的场景。现在做小红书的人这么多，不管你是想做副业、还是本来就在运营账号，我带你看看，Agent 能怎么帮你做小红书运营。

咱们先在 Agent 里装一个现成的技能。你看菜单里有个 Skill 或者技能市场的地方，找到"小红书"相关的那个，点安装。"

【带学员安装小红书 skill】

"装好了，这个技能就等于给你的 Agent 加了一个"小红书运营专家"的大脑。来，咱们用它策划一条笔记。输入——"

> ```
> 我想做小红书，方向是[美食探店 / 穿搭 / 职场干货 / 你的方向]。
> 帮我策划一条能爆的笔记：先给我 5 个选题，挑一个最好的，
> 写出标题、正文文案、话题标签，再给我配图的建议。
> ```

【等结果出来，逐项指着讲】

"你看它给你的——不是一句敷衍的建议，是一整套：5 个选题让你挑、爆款标题、能直接发的正文、带上话题标签，连配几张什么图都告诉你了。

这一套东西，你自己憋，可能得憋一下午。它几十秒就给你了。

而且这还没完，你可以继续压榨它——"

> ```
> 第 3 个标题我最喜欢，帮我再写 3 个不同风格的开头钩子，
> 要那种前 3 秒就让人停下来的
> ```

【等结果】

"看，它又给你换了三个开头。做内容最难的就是选题和开头，这俩它全包了。

我为什么要专门带你做这个？因为这就是很多学员后来靠 AI 赚到第一笔钱的起点。一个能稳定产出内容的账号，本身就值钱；帮别人做这套内容，也有人愿意付费。这个咱们待会儿细说。

你先记住这个感觉：用网页那套逻辑做"作品"，用 skill 这套逻辑做"运营"——同一个 Agent，能干的活儿比你想的多得多。"

## 2.5 小结与钩子

"咱们盘一下这二十多分钟你干了啥：做出了一个真能打开的个人网页；学会了"我说想法、它来实现"这个最核心的用法；还用一个现成技能，做出了一整套能发布的小红书内容。

不错吧？但我猜你心里已经开始琢磨了——"它到底还能做多牛的东西？我这俩是不是只是入门级的？"

问得好。接下来我给你看几个，是这个世界上做到顶尖水准的作品。看完你就知道，你今天摸到的这条路，能通向多远的地方。"

**必念的三句话（重复两遍）：**
1. "以前是会写代码的人才能造东西，现在是会把想法说清楚的人就能造东西。"
2. "你接一单，学费就回来了，还倒赚。"
3. "学习最大的敌人不是难，是过几天就凉了。"

**案例定性口径（重要，别说错）：**
- 这些顶级作品**大多是专业工作室的传统硬功夫**，代表"天花板"。
- 只有 Nithin Warrier（Claude Code）、Pieter Levels（自述）是真·vibe coding。
- 核心话术：**"这些顶级效果，过去是高手专属，现在借助 AI，你也能一步步复刻靠近——重点是学会怎么用它。"**
- 千万别说"这些都是 vibe coding 随便做出来的"，会被懂行的人戳穿。
