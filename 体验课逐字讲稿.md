# 【Part 2】手把手做出你的第一个 Agent 项目

## 2.0 过渡开场（承接安装环节）

【看着大家，语气轻松、带点兴奋】

"好，到这一步，你的 Agent 已经装好了，控制台也登录进去了，刚才那第一条任务也跑通了。我先恭喜你一句——**你现在已经比 90% 的人走得远了。**

为什么这么说？因为'AI Agent'这个词，几乎所有人都听过，抖音上、朋友圈里天天刷到。但你去问一圈，**真正自己动手装好、跑起来、用它做出过东西的人，一百个里都不到十个。** 大多数人停留在'听说过'，而你现在已经'跑起来了'。

不过说实话，装好只是拿到了工具，就像你买了台相机——真正好玩的，是拿它拍出照片。所以接下来这二十几分钟，才是今天最爽的部分。

我先给你交个底：**接下来你一行代码都不用写，一个专业术语都不用背。** 你要做的，就是像跟一个特别能干的助理聊天一样，用大白话告诉它'我想要什么'，然后看着它帮你做出来。

准备好了吗？把你的 Agent 输入框打开，跟着我，我们开始做你人生中的第一个 AI 项目。"

【停顿，确认大家都打开了输入框】

---

## 2.1 抛出目标，制造期待

"我们今天要做的第一个东西，是——**一个完全属于你自己的个人主页网页。**

不是那种填模板、换个头像的东西。是从一张白纸开始，你告诉它你是谁、你喜欢什么风格，它当场帮你写出一个**真能在浏览器打开、真能发给朋友看**的网页。

我带过很多学员做这一步，他们做完第一反应几乎都一样，愣一下，然后说：'就……就这么好了？我啥都没干啊？'

【笑一下】

对，就这么好了。而这，恰恰就是 **AI Agent** 和你以前用的那些聊天 AI **最本质的区别**——

以前的 AI，你问它问题，它给你一段文字回答，剩下的事还得你自己干。
而 Agent 不一样——**它不只是回答你，它真的会动手，帮你把东西做出来。** 它会建文件、写代码、打开给你看。它是一个'会干活的 AI'，不是'会聊天的 AI'。

来，废话不多说，第一句话，跟着我一起打。"

---

## 2.2 第一次对话：让它动手

【放慢，一边念一边让大家跟着输入】

"在输入框里，输入这么一段话——你可以直接照着我的念，也可以把里面的信息换成你自己的，更好。"

【屏幕上打出示范文字，逐句带读】

> ```
> 帮我做一个个人主页网页。我叫[你的名字]，是一名[你的职业或学生身份]，
> 平时喜欢[写两三个兴趣爱好]。我想要[简约现代 / 科技感 / 温暖治愈]的风格，
> 深色背景，页面上要有一个大标题、一段自我介绍、还有我的联系方式。
> 做成一个可以直接用浏览器打开的 HTML 文件。
> ```

"好，把里面的名字、职业、爱好换成你自己的。填好了就发出去。

【学员输入时，讲师引导大家看屏幕】

现在，**盯着屏幕看它接下来做什么**——注意，它不会像以前那样只吐给你一段文字。你会看到它开始'动'起来：它在**创建文件、在一行一行写代码**。就好像有个程序员坐在你旁边，你说了句话，他就埋头开始帮你敲键盘了。

这个过程，有个特别形象的名字，叫 **Vibe Coding**——直译是'凭感觉写代码'。意思就是：**你负责'想要什么'，它负责'怎么实现'。** 你是那个出主意的人，它是那个动手的人。"

【等 Agent 生成完成】

"好，看到它做完了。现在我们让它**直接打开给你看看**。再输入一句——"

> ```
> 在浏览器里帮我打开这个网页看看
> ```

【网页打开的瞬间，讲师提高语气】

"看——！**这是你的网页。** 你自己的名字、你自己的介绍，就在这个页面上。

我想让你意识到一件事：**三分钟之前，这个网页在这个世界上根本不存在。** 是你，刚刚用几句话，把它从'无'变成了'有'。这就是你的第一个 Agent 作品。"

【停顿】

"我想请你现在做一件事——**给它截个图。**

不是为了发朋友圈（当然你想发也可以）。而是因为，这是你和 AI 一起做出来的**第一个东西**，值得你记住这一刻。很多年后你回头看，会发现这是你和 AI 协作的起点。"

---

## 2.3 迭代修改：暴露 Agent 的真正威力

【语气转折，带点"好戏还在后头"的意味】

"但说实话，刚才那个还不算最让我震撼的。真正让我第一次用的时候头皮发麻的，是接下来这个——**修改。**

你觉得网页哪里不满意？**你不用重做，也不用去改代码，你就直接跟它说'我想改哪里'就行。** 来，我们一起改几次，你自己感受。

第一句，我们让标题更有冲击力——"

> ```
> 把标题的字体改大一些，加一个打字机逐字打出来的动画效果
> ```

【等结果出来】

"看到了吗？标题变大了，而且字是一个一个'打'出来的。好，继续，我们让背景动起来——"

> ```
> 背景上加一点会缓慢飘动的粒子，鼠标移动的时候粒子有反应
> ```

【等结果出来】

"哇，背景活了。最后，我们给它加个作品集的位置——"

> ```
> 帮我加一个"作品集"板块，先放三个占位的卡片，以后我好往里填东西
> ```

【等结果出来，然后正色说】

"现在，我要你停下来，认真感受一下刚才发生了什么。

刚才这三次修改——**打字机动画、会动的粒子背景、带交互的卡片布局**——如果你去找一个真人程序员做，这至少是好几个小时的活儿，而且你还得会描述、会验收。

而你呢？**你每次只说了一句'我想要什么'，几秒钟，它就帮你改好了。** 你根本不需要知道什么叫 CSS、什么叫 JavaScript、什么叫动画曲线——这些以前挡在普通人面前的技术门槛，现在**一句话就能跨过去。**

所以我特别想把这句话送给你，也是我自己最深的体会——

**在 AI Agent 的时代，会写代码不再是稀缺能力，会清楚地表达自己想要什么，才是。会表达想法的人，就是能造东西的人。**"

【停顿，让这句话落地】

---

## 2.4 小结与钩子

"我们花一分钟，回顾一下你刚才这二十分钟做成了什么：

第一，你**从零做出了一个真实的、能打开的个人网页**；
第二，你学会了那个最核心的协作方式——**我说想法，它来实现**；
第三，你亲身体验了'改哪说哪'的迭代威力，知道了这东西**能改、能长大**。

这已经很了不起了。但我知道，你现在心里可能已经冒出一个念头：**'那……它还能做多大的东西？我这个只是个小网页而已。'**

问得好。别急——接下来，我要给你看看，**当有人把你今天学的这套方法用到极致，他们做出了什么。** 看完，你就会明白，你今天迈出的这一小步，通向的是一个多大的世界。"

### 💥 案例一（开场炸场）— Pieter Levels 的飞行模拟器

【滚动到 Pieter Levels 案例】

"先看这个，我每次讲到都觉得离谱。

有一个独立开发者，叫 Pieter Levels，在国外挺有名。他有一天就对 AI 说了一句话——**'帮我做一个飞行模拟器'。**

就这一句。然后几个小时之后，一个**完整的 3D 飞行游戏上线了**——注意听，这个游戏里有**多人联机**，好几个人能同时在里面飞；有**空管对话系统**，就是飞机和塔台通话那种；有**天气、有云层、有地形**。一个正经的飞行游戏该有的，它基本都有了。

全网 100 万+ 的曝光。最绝的是作者自己说的一句话，他说：**'我从头到尾，没有完整看过一遍代码。'**

【停顿】

你品一下这句话。他做出了一个百万曝光的游戏，但他没看过自己的代码。因为**代码不是他写的，是 AI 写的，他只负责指挥。** 这就是整个 AI 编程圈里最出圈、最封神的一个时刻。它证明了一件事：**决定你能做出什么的，不再是你会不会写代码，而是你会不会指挥 AI。**"

## 3.2 名校大厂背书：我们凭什么带你

【语气笃定、不夸张，摆事实】

"网上教 AI 的人多如牛毛，凭什么跟我们学？我给你三个实打实的理由，都不吹，你都可以自己去查。

**第一，我们的团队背景。**

我们的团队，来自**清华、腾讯、字节**。请注意我的用词——不是'和这些公司合作过'，是**真的从这些地方出来的人，在亲自带你**。做这门课的核心成员里，有**前字节的专家、前微信 AI 的专家**。这些人平时在做什么？在一线做最前沿的 AI 产品。现在，他们把一线的打法，直接教给你。

**第二，我们不只是'做课的'，我们是真的在教育和研究一线做事的。**

这点很重要。市面上很多 AI 课，是那种'我看了几篇文章、总结几个技巧就来教你'的。我们不是。

我们**和清华合作，办过 Design Future 国际工作坊**，就是教大家用这些 AI 工具来做设计、做创作；我们**办过线下的体验课**，面对面带学生做；我们请过**北京理工大学的教授来做评委**，给学生的作品打分、把关。

这意味着什么？意味着**你今天学的这套教学方法，是经过真实的学术场景、经过高校教授验证过的**，不是我拍脑袋编出来的。它扛得住检验。

**第三，我们自己就在用这套方法做东西。**

你还记得刚才案例里那个用手势控制流体的作品吗？那个清华美院的作品，就是**我们自己团队做的**。

我想说的是：**我们不是那种只会教、自己却做不出东西的老师。我们自己就在一线用这套方法，做出拿得出手的作品。** 我教你的每一招，都是我自己验证过、真的管用的。"

【收束】

"所以，你在我们这儿学的，不是网上随便一个博主总结的零散技巧，而是**清华加大厂的一线打法，一套经过国际工作坊和高校评委验证过的完整体系。** 这是我们最大的底气。"

## 3.3 学了能干嘛：把能力变成钱、变成奖

【语气转向务实、接地气，这是最能打动人的部分】

"好，聊点最现实的。你心里肯定有个问题：**'学了这个，对我到底有什么用？能当饭吃吗？'**

我特别喜欢这个问题，因为这正是我们和别的课**最不一样的地方**——

**别人只教你'怎么用'，我们除了教你，还给你'出口'。** 什么叫出口？就是你学完之后，我们手里有资源，能让你把这个能力**真的变成钱、变成奖。**

我给你讲两条真实的路。

**第一条路：接活赚钱，一单就能翻倍赚回学费。**

我们手上，有**各种企业的合作和外包资源**。你学完这套课，会做网页、会做内容、会做小工具之后，你就**可以通过我们的平台，去接真实的商单**——就是别人付钱，你用 AI 帮他做东西。

我给你算一笔特别实在的账：我们正课的学费，不到 500 块。而在外包市场上，**一个像样的网页、一个企业落地页的单子，报酬就足够你翻倍赚回学费。**

换句话说——**你接一单，学费就回来了，而且还倒赚。** 这不是我给你画饼，这是我们平台上正在真实发生的事情。你的学费，本质上是一笔能快速回本、甚至能持续给你带来收入的投资。

**第二条路：参赛拿奖，打黑客松赢奖金。**

如果你不急着赚钱，你想要的是背书、是荣誉、是能写进简历的东西——我们手里还有**各大高校和初创企业的比赛资源。**

你用 AI 做出来的项目，**我们可以帮你推荐去参加各种比赛**，帮你拿奖、拿背书。我们也会**带你去打黑客松，赢奖金。**

你回想一下刚才那个 Nithin 的网站，用 AI 做的，拿了国际设计大奖提名。你今天做的第一个网页，只要你愿意往下走，**它完全有可能变成一个能拿奖、能写进简历、能帮你谈下实习和工作的作品。**"

【收束】

"所以你看，在我们这儿，学 AI Agent 绝对不是'学个技能，然后就没下文了'。

它是一条完整的链路：**你能靠它接单赚钱，能靠它参赛拿奖，能把它做成副业、甚至事业。** 而我们，是那个**既教你、又给你出口、还帮你把作品推出去**的人。这才是我们真正想给你的东西。"

## 3.4 引导报名正课：临门一脚

【语气放缓，从热烈转向真诚，不催不逼，用逻辑和情绪自然收口】

"好，今天这节体验课，我们快接近尾声了。我们盘一下，短短一节课，你已经完成了这么多事：

你**装好了 Agent**；
你**亲手做出了第一个作品**；
你**看到了这条路能走多远**——从个人网页到国际获奖作品；
你还知道了，**它能帮你赚钱、能帮你拿奖。**

这一节课的价值，我觉得已经远远超过 19 块 9 了。
