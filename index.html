【Part 2】手把手做出你的第一个 Agent 项目

2.0 过渡开场（承接安装环节）

【看着大家，语气轻松、带点兴奋】

"好，到这一步，你的 Agent 已经装好了，控制台也登录进去了，刚才那第一条任务也跑通了。我先恭喜你一句——你现在已经比 90% 的人走得远了。

为什么这么说？因为'AI Agent'这个词，几乎所有人都听过，抖音上、朋友圈里天天刷到。但你去问一圈，真正自己动手装好、跑起来、用它做出过东西的人，一百个里都不到十个。 大多数人停留在'听说过'，而你现在已经'跑起来了'。

不过说实话，装好只是拿到了工具，就像你买了台相机——真正好玩的，是拿它拍出照片。所以接下来这二十几分钟，才是今天最爽的部分。

我先给你交个底：接下来你一行代码都不用写，一个专业术语都不用背。 你要做的，就是像跟一个特别能干的助理聊天一样，用大白话告诉它'我想要什么'，然后看着它帮你做出来。

准备好了吗？把你的 Agent 输入框打开，跟着我，我们开始做你人生中的第一个 AI 项目。"

【停顿，确认大家都打开了输入框】

2.1 抛出目标，制造期待

"我们今天要做的第一个东西，是——一个完全属于你自己的个人主页网页。

不是那种填模板、换个头像的东西。是从一张白纸开始，你告诉它你是谁、你喜欢什么风格，它当场帮你写出一个真能在浏览器打开、真能发给朋友看的网页。

我带过很多学员做这一步，他们做完第一反应几乎都一样，愣一下，然后说：'就……就这么好了？我啥都没干啊？'

【笑一下】

对，就这么好了。而这，恰恰就是 AI Agent 和你以前用的那些聊天 AI 最本质的区别——

以前的 AI，你问它问题，它给你一段文字回答，剩下的事还得你自己干。
而 Agent 不一样——它不只是回答你，它真的会动手，帮你把东西做出来。 它会建文件、写代码、打开给你看。它是一个'会干活的 AI'，不是'会聊天的 AI'。

来，废话不多说，第一句话，跟着我一起打。"

2.2 第一次对话：让它动手

【放慢，一边念一边让大家跟着输入】

"在输入框里，输入这么一段话——你可以直接照着我的念，也可以把里面的信息换成你自己的，更好。"

【屏幕上打出示范文字，逐句带读】

帮我做一个个人主页网页。我叫[你的名字]，是一名[你的职业或学生身份]， 平时喜欢[写两三个兴趣爱好]。我想要[简约现代 / 科技感 / 温暖治愈]的风格， 深色背景，页面上要有一个大标题、一段自我介绍、还有我的联系方式。 做成一个可以直接用浏览器打开的 HTML 文件。

"好，把里面的名字、职业、爱好换成你自己的。填好了就发出去。

【学员输入时，讲师引导大家看屏幕】

现在，盯着屏幕看它接下来做什么——注意，它不会像以前那样只吐给你一段文字。你会看到它开始'动'起来：它在创建文件、在一行一行写代码。就好像有个程序员坐在你旁边，你说了句话，他就埋头开始帮你敲键盘了。

这个过程，有个特别形象的名字，叫 Vibe Coding——直译是'凭感觉写代码'。意思就是：你负责'想要什么'，它负责'怎么实现'。 你是那个出主意的人，它是那个动手的人。"

【等 Agent 生成完成】

"好，看到它做完了。现在我们让它直接打开给你看看。再输入一句——"

在浏览器里帮我打开这个网页看看

【网页打开的瞬间，讲师提高语气】

"看——！这是你的网页。 你自己的名字、你自己的介绍，就在这个页面上。

我想让你意识到一件事：三分钟之前，这个网页在这个世界上根本不存在。 是你，刚刚用几句话，把它从'无'变成了'有'。这就是你的第一个 Agent 作品。"

【停顿】

"我想请你现在做一件事——给它截个图。

不是为了发朋友圈（当然你想发也可以）。而是因为，这是你和 AI 一起做出来的第一个东西，值得你记住这一刻。很多年后你回头看，会发现这是你和 AI 协作的起点。"

2.3 迭代修改：暴露 Agent 的真正威力

【语气转折，带点"好戏还在后头"的意味】

"但说实话，刚才那个还不算最让我震撼的。真正让我第一次用的时候头皮发麻的，是接下来这个——修改。

你觉得网页哪里不满意？你不用重做，也不用去改代码，你就直接跟它说'我想改哪里'就行。 来，我们一起改几次，你自己感受。

第一句，我们让标题更有冲击力——"

把标题的字体改大一些，加一个打字机逐字打出来的动画效果

【等结果出来】

"看到了吗？标题变大了，而且字是一个一个'打'出来的。好，继续，我们让背景动起来——"

背景上加一点会缓慢飘动的粒子，鼠标移动的时候粒子有反应

【等结果出来】

"哇，背景活了。最后，我们给它加个作品集的位置——"

帮我加一个"作品集"板块，先放三个占位的卡片，以后我好往里填东西

【等结果出来，然后正色说】

"现在，我要你停下来，认真感受一下刚才发生了什么。

刚才这三次修改——打字机动画、会动的粒子背景、带交互的卡片布局——如果你去找一个真人程序员做，这至少是好几个小时的活儿，而且你还得会描述、会验收。

而你呢？你每次只说了一句'我想要什么'，几秒钟，它就帮你改好了。 你根本不需要知道什么叫 CSS、什么叫 JavaScript、什么叫动画曲线——这些以前挡在普通人面前的技术门槛，现在一句话就能跨过去。

所以我特别想把这句话送给你，也是我自己最深的体会——

在 AI Agent 的时代，会写代码不再是稀缺能力，会清楚地表达自己想要什么，才是。会表达想法的人，就是能造东西的人。"

【停顿，让这句话落地】

2.4 小结与钩子

"我们花一分钟，回顾一下你刚才这二十分钟做成了什么：

第一，你从零做出了一个真实的、能打开的个人网页；
第二，你学会了那个最核心的协作方式——我说想法，它来实现；
第三，你亲身体验了'改哪说哪'的迭代威力，知道了这东西能改、能长大。

这已经很了不起了。但我知道，你现在心里可能已经冒出一个念头：'那……它还能做多大的东西？我这个只是个小网页而已。'

问得好。别急——接下来，我要给你看看，当有人把你今天学的这套方法用到极致，他们做出了什么。 看完，你就会明白，你今天迈出的这一小步，通向的是一个多大的世界。"

💥 案例一（开场炸场）— Pieter Levels 的飞行模拟器

【滚动到 Pieter Levels 案例】

"先看这个，我每次讲到都觉得离谱。

有一个独立开发者，叫 Pieter Levels，在国外挺有名。他有一天就对 AI 说了一句话——'帮我做一个飞行模拟器'。

就这一句。然后几个小时之后，一个完整的 3D 飞行游戏上线了——注意听，这个游戏里有多人联机，好几个人能同时在里面飞；有空管对话系统，就是飞机和塔台通话那种；有天气、有云层、有地形。一个正经的飞行游戏该有的，它基本都有了。

全网 100 万+ 的曝光。最绝的是作者自己说的一句话，他说：'我从头到尾，没有完整看过一遍代码。'

【停顿】

你品一下这句话。他做出了一个百万曝光的游戏，但他没看过自己的代码。因为代码不是他写的，是 AI 写的，他只负责指挥。 这就是整个 AI 编程圈里最出圈、最封神的一个时刻。它证明了一件事：决定你能做出什么的，不再是你会不会写代码，而是你会不会指挥 AI。"

💥 案例三（关键证据）— Nithin Warrier，明确标注 Claude Code 构建

【滚动到 Nithin Warrier 案例】

💥 案例四（自己人，拉近距离）— Evans 手势流体（清华美院）

从你今天做的第一个个人网页，到清华正在做的这种前沿交互研究——中间隔着的，不是一道你跨不过去的天堑，而是一条你今天已经踏上去的路。 起点是一样的，都是'跟 AI 说你想要什么'。区别只在于，你愿意在这条路上走多远。"

3.2 名校大厂背书：我们凭什么带你

3.4 引导报名正课：临门一脚

【语气放缓，从热烈转向真诚，不催不逼，用逻辑和情绪自然收口】

"好，今天这节体验课，我们快接近尾声了。我们盘一下，短短一节课，你已经完成了这么多事：

你装好了 Agent；
你亲手做出了第一个作品；
你看到了这条路能走多远——从个人网页到国际获奖作品；
你还知道了，它能帮你赚钱、能帮你拿奖。

这一节课的价值，我觉得已经远远超过 19 块 9 了。

但我要跟你说实话：今天，我们只带你'跑通'了第一步。

你现在手里的，还只是一个存在你自己电脑上的小网页。它还没上线，别人打不开；你的 Agent 还只能在这台电脑上用，你一关机它就停了。

真正的威力，在正课里。我给你说说正课和今天的区别——

在正课里，我们会带你：
